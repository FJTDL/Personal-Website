Statistical Methods in Analysing Mice Migration in Abel Tasman National Park

Building on a project from STATS 399 - Capstone, the objective of this analysis is to determine if the 2007 extermination events on the Tonga and Adele islands were successful, the implications of this, and what further insights can be gained from the data.

Techniques

  • Partial Least Squares - Discriminant Analysis (PLS-DA)
  • Hamming Distance Hierarchical Clustering
  • Permutation Testing

Preamble

Firstly, I found the original theory of using Principle Component Analysis (PCA) to be a very interesting idea, and that it did work as intended, allowing us to investigate the origins of the mice populations.

However, when it comes to trying to differentiate between groups, LDA is a more standard technique. LDA has very stringent requirements that are rarely met, and QDA while less stringent, it also still has requirements in order to work properly. Although I could have checked if the requirements were met, I chose to use a non-parametric method instead, skipping over these requirements, and drawing results from a PLS-DA model.

Additionally, since we are considering genetic data, it would make sense to try a tree model, akin to phylogenetic trees. This is easily done using the base R dist() and hclust() functions, but the problem was that the distances programmed for the base methods do not include Hamming distance, which would be the preferred measure in this problem.

In this web-report, you will find explanations to my methods and analyses, as well as my conclusions. However, this will, in all likelihood, end up being a shortened version without much of the statistical depth I would usually include in a report. Why? Because it makes this page horribly long, and you really don't need it.

PLS-DA

When we plot all of the populations, we can see a rather clear overlap of the groups, most significantly with the mainland group. There isn't much more we can learn from this plot right now so, let's remove the mainland!

By removing the mainland, we can now more clearly see some separation between the groups. For example, it currently appears that the Tonga15 population is very distinct from the Tonga07, while the Fisherman island population is almost a complete subset of the Adele07 population.

Here, we can see that the mainland population and the Adele15 are covering almost the exact same space, while the Fisherman population appears to overlap significantly with all other populations.

We can see from this plot that the Adele populations both overlap considerably with the Mainland, as well as each other. However, the direction that the two Adele populations are facing is different, suggesting that they are more similar to the Mainland than to each other.

We can quite clearly see that the Adele15 population is far more varied, and that it doesn't overlap as much with the Adele07 population as much as the previous plots might have suggested. This indicates that the two populations are not that closely related.

It is worth nothing that in all of these analyses, the PLS-DA method captures very low levels of variation. Even when the method is providing a high level of separation visually, it is important to remember that these plots do not even account for 25% of the total variation seen in the data.

Overall, these plots tell us that there is fairly strong evidence to suspect that all populations are descended from the mainland in some way, that the Adele 2015 and Tonga 2015 populations are likely to be recent invaders from the mainland rather than survivors, and that Fisherman island 2007 population was likely seeded by Adele island rats sometime prior to the 2007 eradication event.

Ideally, what we want to do now is find a way to compare how similar the rats are, and use this to construct a clustering solution, similar to phylogenetic trees. Given we are working with genetic data, this is a logical progression, and using the dist() and hclust() functions, this should be a fairly simple process.

However, one important detail is that the variables themselves are numbers, but these numbers are labels, not actual numeric values. This means that using a distnace such as Manhattan or Euclidean, would give a false idea of their similarity. A binary method would be better, but the base R binary distance checks if two observations are exactly the same, when we want to check this on a column-wise basis.

Thankfully, there is a well known distance we can easily implement to fix this, called the Hamming Distance, which we can implement easily with the function below.

Hamming Distance Function

      
        match_dist <- function(x, y) {
          sum(x != y) 
        }

This does, however, mean that we also need to install and load the Proxy package, so be aware of this if you attempt to recreate my code.
Additionally, note that the red clusters are somewhat arbitrary, and have mostly been used for illustrative purposes rather than forming proper clusters.

Hierarchical Clustering Solutions.

The full cluster solution shows a distinct branch of Fisherman island that are most closely related to a branch of mostly Adele07 mice, both of which are far removed from the other branches. We also observe a clear cluster of Tonga07 mice, also rather distinct of other groups, while the Mainland and 2015 populations all appear to cluster together in roughly the same branches.

Considering the Adele populations with the Fisherman and Mainland populations, we see quite clearly the Adele07 and Fisherman populations form distinct clusters, and are most closely related to each other, and reasonably removed from the Mainland and Adele15 populations, which seem to cluster with each other. Adele15 forms several smaller clades within the larger cluster instead of clustering together in large groups as the other populations tend to do.

When considering just the Adele and Mainland populations, we see very clearly that the Adele07 population forms a distinct cluster by itself, while the Adele15 mice tend to "mix" with the Mainland population, seemingly randomly.

Comparing the 2015 Adele and Tonga populations with the Mainland population, we see almost perfect mixing between the populations, indicating that these populations are not distinct of each other.

In regards to Tonga, we once again we observe a 2007 population as being highly distinctive, while the 2015 "mixes" with the mainland. However, we observe much less scattering of the 2015 Tonga population compared to Adele, and in particular observe a distinctive clade emerge within one of the clusters. This suggests that the Tonga population may be old and thus more established, or that migration to the island is less common than it seems to be with Adele.

So what do we learn from these clustering solutions?

  • There is strong indication that the Fisherman07 Population is likely a "spin off" of the Adele07 population.
  • There is strong evidence to suggest that the 2007 eradication events were successful on all islands.
  • It is likely that the Adele15 population is the result of either multiple invasions, or constant flow of migrating rats from the mainland.
  • The 2015 Populations are not yet sufficiently established to be distinct of the mainland lineage.
  • Tonga island appears to have less frequent migration from the Mainland.

Permutation Testing

One advantage to this distance-based method is that we can compare the distances between two groups for statistical significance, and draw a more robust conclusion about the origin of the mice. However, many of the graphical methods proved viable in finding the likely conclusions, so we do not need to apply many of these. In fact, we shall only apply one, to try to answer the question of if the Fisherman island population really is a "spin-off" of the Adele07 population, or if it came from the mainland.


Permutation Testing

    
      adele7 <- grouped_mainland %>%
        filter(
          pop == "Adele07"
        )

      adele15 <- grouped_mainland %>%
        filter(
          pop == "Mainland"
        )

      adele7_geno <- adele7[, -(1:2)]
      adele15_geno <- adele15[, -(1:2)]

      dist_matrix <- as.matrix(dist(rbind(adele7_geno, adele15_geno)), method = match_dist)

      n_f <- nrow(adele7_geno)
      n_a <- nrow(adele15_geno)

      adele7_idx <- 1:n_f
      adele15_idx <- (n_f + 1):(n_f + n_a)

      pairwise_distances <- dist_matrix[adele7_idx, adele15_idx]
      a_a_distance <- mean(pairwise_distances)

      obs_diff <- a_a_distance - a_f_distance

      subset_data <- subset(grouped_mainland, pop %in% c("Fisherman07", "Adele07", "Adele15"))
      geno_only <- subset_data[, -(1:2)]  # remove ID/Pop columns
      pop_labels <- subset_data$pop


      n_perm <- 10000
      perm_diff <- numeric(n_perm)

      set.seed(123)

      for (i in 1:n_perm) {
        shuffled <- sample(pop_labels)

        fisherman_idx <- which(shuffled == "Fisherman07")
        adele07_idx   <- which(shuffled == "Adele07")
        adele15_idx   <- which(shuffled == "Adele15")
        f_a_dist_perm <- mean(as.matrix(dist(geno_only, method = match_dist))[fisherman_idx, adele07_idx])
        a_a_dist_perm <- mean(as.matrix(dist(geno_only, method = match_dist))[adele07_idx, adele15_idx])

        perm_diff[i] <- a_a_dist_perm - f_a_dist_perm
      }

      p_value <- (sum(perm_diff >= obs_diff) + 1) / (n_perm + 1)

      p_value

Conclusions

Many of the conclusions drawn have been listed in prior sections, so consider this a summary.
We can be quite confident that the 2007 eradication events were successful, and that the 2015 populations were seeded from the mainland. At the same time, we can also be quite confident that the Fisherman island 2007 population was seeded by the Adele 2007 population. This means that while the eradication method works, as long as there are mice on the mainland, they will get on the offshore islands as well, and may spread to other islands they may not have swum to from the mainland due to distance issues, using the immediate offshore islands as "staging grounds". There also appears to possibly be something about Tonga island that discourages a constant flow of mice to the island, although this could simply be due to chance.
